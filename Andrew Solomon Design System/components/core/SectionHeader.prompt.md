# SectionHeader

The top-level section marker — centered, bold, amber, lightly tracked (0.04em). It marks Summary, Technical Skills, Experience, Education, Certifications. This is the primary place the accent appears in running content, so keep labels to one or two words.

```jsx
<SectionHeader>Experience</SectionHeader>
<SectionHeader align="start" as="h3">Skills</SectionHeader>
```

Props: `children`, `align` (`center` | `start`), `as` (heading tag).
