# Hairline

The one divider primitive — a 1px whisper-grey rule, full width. It carries every horizontal division in the system. No thicker borders, no filled separator bars, no boxes. If a division feels weak, add whitespace, never weight.

```jsx
<Hairline />
<Hairline spacing="lg" />
```

Props: `spacing` (`none` | `sm` | `md` | `lg` | `xl`, default `md`).
