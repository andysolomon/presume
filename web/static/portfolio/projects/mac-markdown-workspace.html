Case Study

Mac Markdown Workspace

A polished Markdown workspace with source editing, split preview, WYSIWYG mode, Mermaid diagrams, math rendering, themes, and export tooling — one React codebase shipping to macOS desktop, the browser, and iOS.

Live Demo

Problem

Build a local-first Markdown product experience that feels like a desktop workspace, while reaching the browser and mobile from the same code — so a single editor ships to macOS, the web, and iOS without a rewrite per platform.

Implementation Highlights

  • One platform-agnostic React app targeting Electron, web, and iOS through a single host-capability abstraction
  • Secure local-file boundaries: sandboxed renderer, all file access behind a typed API
  • Source editor with counters and editing modes
  • Rich markdown preview with Mermaid, emoji, and math support
  • Editable preview mode for direct content updates
  • Theme system (light/dark/system) and keyboard shortcut workflow
  • Mobile-responsive UI tuned for iOS (safe-area insets, no zoom-on-focus, no horizontal scroll)
  • Export support paths for TXT, PDF, and DOCX

Current Status

  • macOS desktop app build complete
  • Web app deployed on Vercel for live demonstration
  • iOS build wired via Capacitor
  • Lint, typecheck, unit tests, web build, and iOS build passing in project checks
